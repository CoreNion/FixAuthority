# FixAuthority
このバッチは、何らかの原因で権限が壊れてしまったファイルやフォルダーを直すツールです  
Windowsの元の権限に修復します。
### 利用例
- 他OSとファイルを共有していたら権限のエラーが出た  
- Windowsのインストール・アップデートに失敗しておかしくなった
# How use 
batを起動します(管理者権限で起動しなくても、自動的に管理者権限で再起動されます)  
あとは画面の指示に従って実行するだけです。
